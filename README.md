#  Network Fault Tollerance Tic-Tac-Toe
Features:
* Server - based on epoll()
* Client - based on poll()
* Graphics - based on psedo-graphics
* Supported os - Linux
HOWTO:
* build - make
* run server - ./server [port] [state] (port : 2345, 2346, 2347: state: 0-main, 1-reserve)
* run cline - ./client
